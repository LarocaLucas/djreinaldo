## Sobre o Site

### Design
- Estética editorial — grafite e dourado
- Fontes: **Playfair Display** (display) + **DM Sans** (corpo)
- Layout split two-column no hero
- Numeração lateral nas seções
- Galeria masonry com 26 fotos

### Estrutura
```
/
├── index.html
├── css/
│   └── style.css
├── js/
│   └── main.js
└── assets/
    └── images/
        ├── logo.png
        ├── foto-hero.jpg
        └── galeria/
            └── foto-001.jpg … foto-026.jpg
```

### Funcionalidades
- Navegação fixa com scroll suave
- Animações de entrada (reveal on scroll)
- Botão flutuante WhatsApp
- Galeria com efeito hover
- Responsivo para mobile
